# Lyric Hayes Basketball Strength Tracker

This is a self-contained workout tracker for a 16-year-old basketball player who needs a cautious, back-aware plan because of scoliosis.

## Repository details

Repository name: `lyric-basketball-strength-tracker`

Description: A back-aware basketball strength and conditioning tracker for Lyric Hayes, with a bigger leg and arm focus, 30-minute workouts, optional under-45-minute add-ons, cardio, safety notes, and progress logging.

## What is included

- 3 workouts per week
- 6-week tracker
- 30-minute default sessions
- Optional add-ons that keep sessions under 45 minutes
- Leg strength, arm strength, beginner plyometric vertical-jump mechanics, core control, basketball-friendly cardio, and cool-down work
- StrengthFit-style calendar with planned Workout A/B/C days
- Progress tab with weekly stats, next workout, logged minutes, and next-step guidance
- Exercise Description tab with muscles worked, basketball carryover, and back-care cues
- Back pain and effort logging
- CSV export from the browser
- Print-friendly layout

## How to use

Open `index.html` in a browser. Progress is saved in that browser's local storage.

Suggested schedule:

- Monday: Workout A
- Wednesday: Workout B
- Friday: Workout C

Or use Tuesday, Thursday, Saturday if that fits the week better.

## Medical note

This plan is not medical care. Lyric's pediatrician, orthopedist, or physical therapist should clear the plan, especially if she has brace instructions or recent pain. Stop a workout for sharp pain, nerve symptoms, dizziness, or back pain above 2 out of 10.
